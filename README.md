# P&L Dashboard - Amazon Seller Analytics

A comprehensive React dashboard for tracking profitability and performance metrics across multiple Amazon seller brands.

## Features

- **Executive Summary**: KPIs, alerts, and portfolio-level trends
- **P&L Waterfall**: Visual breakdown of revenue to profit
- **Brand Deep Dive**: Individual brand performance analysis
- **Advertising Intel**: TACoS, ACoS, and ad spend tracking
- **Fee Forensics**: Detailed fee analysis (FBA, storage, referral)
- **Gaps & Recommendations**: Actionable insights

## Brands Tracked

- Fomin
- House of Party
- Functions Labs
- Soul Mama
- Roofus Pet
- Rockport Tools
- Custom Products

## Tech Stack

- React 18
- Recharts for data visualization
- Custom dark theme UI

## Getting Started

```bash
npm install
npm start
```
